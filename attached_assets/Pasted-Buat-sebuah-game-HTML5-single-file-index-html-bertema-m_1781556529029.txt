Buat sebuah game HTML5 single-file (index.html) bertema memancing yang sederhana namun menarik dan dapat dimainkan langsung di browser tanpa memerlukan server atau library eksternal.
Konsep gameplay:
Pemain mengontrol kail pancing dengan klik atau tap.
Tekan dan tahan untuk menurunkan kail ke dalam air, lalu lepaskan untuk menariknya kembali.
Jika kail menyentuh ikan saat ditarik, pemain mendapatkan poin.
Ikan bergerak dari kiri ke kanan atau kanan ke kiri dengan kecepatan acak.
Tampilkan minimal 3 jenis ikan dengan ukuran dan nilai poin berbeda.
Tambahkan timer 60 detik dan tampilkan skor secara real-time.
Setelah waktu habis, tampilkan layar Game Over beserta skor akhir dan tombol Play Again.
Desain:
Gunakan grafis sederhana berbentuk 2D dengan warna cerah.
Latar terdiri dari langit, laut, dan dasar laut.
Tambahkan animasi ombak ringan agar permainan terasa hidup.
Efek suara opsional menggunakan Web Audio API tanpa aset eksternal.
Teknis:
Semua HTML, CSS, dan JavaScript berada dalam satu file index.html.
Tidak menggunakan framework atau library pihak ketiga.
Responsif untuk desktop dan mobile.
Kode rapi, diberi komentar, dan mudah dimodifikasi.
Performa ringan sehingga cocok dijadikan demo atau dipublikasikan di platform HTML5.
Fitur tambahan (opsional):
Sistem combo jika berhasil menangkap beberapa ikan berturut-turut.
Ikan langka yang muncul sesekali dengan poin besar.
Efek partikel kecil saat ikan berhasil ditangkap.
Penyimpanan skor tertinggi menggunakan localStorage.
